# Installation

สร้างไฟล์ .env โดยมีรายละเอียด ดังนี้

```
DB_HOST=localhost
DB_PORT=3306
DB_NAME=hosxp
DB_USER=root
DB_PASSWORD=xxxxx
```

จากนั้น clone project ดังนี้

```
git clone https://github.com/siteslave/SmartIPD-Api.git

cd SmartIPD-Api

npm install

node bin/www
```

เรียกใช้งาน `http://localhost:3000`
